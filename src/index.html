How it Works


Blockchain technology has the potential to transform the storage and accessing of patient Electronic Health Records from its current state to an efficient patient-oriented medical service. This guarantees enhanced security, privacy and control of the records.

Since blockchain makes the system decentralized, it is virtually impossible for the data to get lost since every node in the network has it, as opposed to one centralized database

Once the data is uploaded into the blockchain, the modification or tampering of data is nearly impossible, which ensures the security of sensitive data like medical records.
